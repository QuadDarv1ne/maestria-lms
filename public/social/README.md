# Social Media Images

Place your social media images in this directory:

## Required Images:

1. **og-image.png** (1200x630px)
   - Used for Open Graph / Facebook / LinkedIn shares
   - Recommended: 1200x630 pixels (1.91:1 ratio)

2. **og-image-small.png** (512x512px)
   - Fallback for smaller previews
   - Can be the same as your logo

3. **twitter-image.png** (1200x600px)
   - Used for Twitter Card shares
   - Recommended: 1200x600 pixels

## Current Status:

This is a placeholder directory. Add your actual image files to enable social media sharing previews.

## Design Tips:

- Use high contrast for better visibility
- Include your logo
- Add a clear headline
- Keep text minimal and readable
- Use brand colors (#6366f1 for Maestria)
