# PWA Icons

Place your PWA icons in this directory with the following naming convention:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png` (required for Android)
- `icon-384x384.png`
- `icon-512x512.png` (required)

## Recommended:

1. Create a 512x512 PNG icon with transparent background
2. Use tools like [https://realfavicongenerator.net](https://realfavicongenerator.net) to generate all sizes
3. Ensure the icon has good contrast and visibility at small sizes
